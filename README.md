# Drone Telemetry Analyzer

A Python-based telemetry analysis system designed to process and analyze drone flight data.
This project focuses on flight safety metrics such as altitude, speed, battery level, and temperature.

## Features
- Analyze drone flight telemetry data
- Detect critical conditions (low battery, high temperature)
- Modular and scalable Python architecture
- CLI-based output for fast analysis

## Technologies Used
- Python 3
- CSV data processing
- Modular programming principles
- Git & GitHub
